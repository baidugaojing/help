示例 - bash 需要把 $SERVICEKEY 和 $SERVICE_ID 换为在百度告警平台中创建的服务
参数说明 参数名类型必须说明 appid string 是 应用id，由百度告警系统管理员分配，需放入http头部 token string 是 应用访问所需的token，由百度告警系统管理员分配，需放入http头部 signature string 是 request签名，需放入http头部 useid int 是 应用消费消息配额的user信息，默认为0 需放入http头部 channel enum('sms', 'phone', 'email') 是 推送消息的类型 description string 是 推送消息的内容，channel='sms'|'email'时有效 title string 否 推送消息的标题，channel='sms'时有效 receiver string 是 channel='sms'|'phone'时请填写手机号码，channel='email'时请填写邮箱地址
URL - POST http://gaojing.baidu.com/event/create
告警平台提供API接入方式，有如下两类API接入推送事件API：推送事件API：通过该API接入的数据，可使用告警平台的全部功能，如故障管理等推送消息API：通过该API可以直接发送消息给指定的号码
